# Pandas Data Analysis

## Description

This project demonstrates basic data analysis using the Pandas library and data visualization using Matplotlib.

## Features

* Load data from a CSV file
* Calculate the average of the Marks column
* Create a Bar Chart
* Create a Scatter Plot
* Create a Heatmap

## Technologies Used

* Python
* Pandas
* Matplotlib

## Files

* `main.py` - Python source code
* `students.csv` - Dataset
* `requirements.txt` - Required Python libraries

## How to Run

1. Install the required libraries:

```bash
pip install -r requirements.txt
```

2. Run the program:

```bash
python main.py
```
